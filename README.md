# Cadastro de Alunos – Trabalho AV2 de PCA

**Descrição breve:**\
Sistema em Python para cadastrar, pesquisar, editar e remover alunos, gravando tudo em CSV usando pandas.

**Alunos:**\
• Daniel Moraes

---
**Como executar:**
```
pip install pandas
python main.py
```
---
**Funcionalidades:**\
• Inserir aluno\
• Pesquisar por nome ou matrícula\
• Editar aluno\
• Remover aluno\
• Salvar em CSV\
• Modularização com funções\
• Uso de lista, dicionário e pandas DataFrame
